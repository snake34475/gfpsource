# 故事动画系统完整文档

## 一、概述

本项目是一个基于 **ActionScript 3 (Flash)** 的游戏客户端，故事动画资源使用 **SWF** 格式，通过 `ClientConfig.getCartoon()` 方法从远程服务器动态加载。

**资源基础 URL**: `http://gf.61.com/res/cartoon/`

**完整 URL 规则**: `http://gf.61.com/res/cartoon/{动画名称}.swf`

**直接访问方式**: 在浏览器中打开上述 URL 即可下载 SWF 文件

---

## 六、动画加载与播放流程

```
1. 任务系统触发
   └── TaskXML_StoryAnimation 接收任务触发

2. 反射实例化动画类
   └── 通过类名动态创建 StoryAnimationTask_<ID> 实例

3. 加载 SWF 资源
   └── ClientConfig.getCartoon("动画名称") → 生成完整 URL
   └── UILoader.load() → 从服务器下载 SWF

4. 播放动画
   └── MovieClip.gotoAndPlay(1)
   └── 添加到 LayerManager.topLevel 显示层
   └── 绘制黑色背景遮罩

5. 监听完成
   └── ENTER_FRAME 事件检测是否到达最后一帧
   └── 或用户点击关闭/跳过按钮

6. 清理与回调
   └── 移除 MovieClip、背景、按钮
   └── 恢复背景音乐
   └── 分发 TASK_ANIMATION_START / TASK_ANIMATION_FINISH 事件
   └── 推进任务进度
```

## 七、动画类命名规则

文件命名: `StoryAnimationTask_<任务ID>_<变体编号>.as`

- **任务ID**: 对应游戏中的任务编号
- **变体编号**: 同一任务的多个动画片段 (0, 1, 2, 3...)

示例:
- `StoryAnimationTask_2064_1.as` → 任务 2064 的第 1 个动画片段
- `StoryAnimationTask_2080_4.as` → 任务 2080 的第 4 个动画片段

## 九、技术细节

### 9.1 目标分辨率
- 宽度: 960px
- 高度: 560px

### 9.2 显示层级
- 动画播放于 `LayerManager.topLevel`
- 背景遮罩覆盖屏幕边缘，居中显示动画

### 9.3 交互控制
- 关闭按钮: SWF 内部的 `closeBtn` 元件
- 跳过按钮: 可选，位于右下角
- 点击关闭/跳过/播放完毕均触发 `onFinish()`

### 9.4 音效
- 使用 `ListSoundPlayer` 播放动画音效
- 动画播放期间暂停背景音乐
- 动画结束后恢复背景音乐

### 9.5 主线时间线标记
- `StoryAnimationTaskBaseLoader.as` 包含 `_isMainTimeLine` 标记
- 用于区分主线动画与普通动画的播放行为
